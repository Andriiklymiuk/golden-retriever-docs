# `golden upgrade`

Self-update the CLI via whichever method you installed with (Homebrew or the shell/PowerShell installer). Alias: `golden update`.

```bash
golden upgrade
# or
golden update
```

It detects your install method and runs the right update path, so you don't have to remember whether it was `brew` or the installer.

Installed [from source with cargo](./install.md)? Re-run the `cargo install --git …` command to update.
